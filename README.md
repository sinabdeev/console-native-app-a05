# Console Native App A05

This is a Kotlin/Native console application that demonstrates basic functionality including:
- Displaying a welcome message
- Detecting the operating system
- Showing current date/time and day of week
- Playing a beep sound

## Features

- Cross-platform: Works on Windows, Linux and macOS
- Shows system information
- Demonstrates basic Kotlin/Native capabilities

## Usage

Simply run the application. It will:
1. Display a welcome message
2. Show the operating system it's running on
3. Display the current date/time and day of week
4. Play a beep sound

## Development

The project uses Kotlin Multiplatform and is configured for native compilation. It includes basic interop capabilities for platform-specific functionality.

## Build Output

After building the project, you can find the executable in the following location:
```
build/bin/native/debugExecutable/MyNativeApp.exe
```
